在脑电（EEG）采集中，这种由于电极接触不良、松动或瞬间脱落产生的异常大数值，在信号处理中被称为**脱落伪迹 (Electrode-Pop Artifact)** 或 **设备截断 (Clipping)**。

**必须尽早处理掉这些无效数据，否则它们会造成灾难性的影响！**

### 一、 为什么这些异常值影响极其恶劣？

1. **对滤波操作的影响（振铃效应）：**

   正常的脑电信号幅度通常在 $\pm10\mu V$ 到 $\pm100\mu V$ 之间波动。如果信号突然跳变到 `374.999...`，这在时域上相当于一个“阶跃信号”或“脉冲信号”。当你把包含这种跳变的信号送入 50Hz 陷波器或带通滤波器时，会引发滤波器的**振铃效应 (Ringing Effect)**。滤波器不仅无法有效滤除它，反而会在这个突变点的前后产生剧烈的、持续好几秒的人造震荡波（形似水波纹）。这会把原本干净的相邻数据也彻底污染。

2. **对模型分析的影响（特征爆表）：**

   你后续要计算频段能量 (Energy) 和微分熵 (DE)。这种极端的数值跳变包含了所有频段的极高能量（表现为极宽的频谱）。如果你不剔除它们，算出来的某几秒的 DE 特征会比正常值大几百倍。你的深度学习模型或随机森林会误以为这是一种“极其强烈的认知活动”，从而把噪声当作核心特征去学习，导致模型在测试集上彻底失效。

### 二、 如何在代码中解决这个问题？

处理这种电极接触不良的数据，业界标准的做法分为两步：**滤波前插值修复（防震荡）** + **切窗时直接丢弃（保质量）**。

我们需要在之前的 `preprocess.py` 和 `feature_extraction.py` 中增加异常值检测机制。

#### 步骤 1：在滤波前修复/限制异常值 (防止滤波器崩溃)

在信号送入滤波器之前，我们需要识别出这些异常值。如果只是瞬间的松动（几个采样点），我们可以用前后的正常数据把它“平滑插值”掉；如果是一大片，我们先把它限制在一个合理的生理边界内。

请在你的 `EEGPreprocessor` 类中加入以下方法，并在执行基础滤波**之前**调用它：



```python
    def handle_extreme_values(self, data: np.ndarray, threshold: float = 300.0) -> np.ndarray:
        """
        处理超出正常生理范围的极端异常值（电极接触不良）
        :param data: 一维或二维的脑电数据
        :param threshold: 异常值判定阈值 (微伏)。一般前额脑电很少超过 ±150 
        """
        clean_data = data.copy()
        
        # 遍历每一个通道
        for i in range(clean_data.shape[1] if clean_data.ndim > 1 else 1):
            channel_data = clean_data[:, i] if clean_data.ndim > 1 else clean_data
            
            # 找到超出阈值的点的索引 (例如大于150或小于-150，或者像你的374)
            outlier_mask = np.abs(channel_data) > threshold
            
            if np.any(outlier_mask):
                outlier_count = np.sum(outlier_mask)
                print(f"通道 {i}: 发现 {outlier_count} 个极端异常采样点，正在进行插值修复...")
                
                # 创建一个 pandas Series 利用其强大的插值功能
                s = pd.Series(channel_data)
                
                # 将异常值替换为 NaN
                s[outlier_mask] = np.nan 
                
                # 1. 首先尝试线性插值 (适用于短暂脱落)
                # limit=25 表示如果连续超过25个点(0.1秒)都是异常的，就不插值了
                s = s.interpolate(method='linear', limit=25, limit_direction='both')
                
                # 2. 如果还有大片连续的 NaN (说明电极彻底掉了很久)，
                # 只能进行前向/后向填充或者将其限制在阈值边界，防止后续滤波器报错
                s = s.fillna(method='ffill').fillna(method='bfill') 
                
                # 最后做一次硬裁剪，确保绝对没有超出阈值的数据进入滤波器
                s = s.clip(lower=-threshold, upper=threshold)
                
                if clean_data.ndim > 1:
                    clean_data[:, i] = s.values
                else:
                    clean_data = s.values
                    
        return clean_data
```
